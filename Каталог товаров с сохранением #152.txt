Каталог товаров с сохранением #152
В этом уроке надо будет написать каталог товаров с сохранением введенных данных в текстовый файл.

В программе должен быть создан пустой словарь — catalog. С помощью цикла программа должна предложить ввести пользователю три товара. В каждой итерации цикла должно вводиться два значения — наименование и количество товара. Наименование и количество должны вводится с помощью двух разных вызовов функции input.

Количество товара после ввода преобразуйте в тип int. Правильность ввода пользователя проверять не нужно. Понадеемся, что он будет вводить только числа.

Каждое наименование и количество надо добавить в словарь catalog. Наименование должно быть ключом, а количество — значением этого ключа.

Если пользователь введет несколько одинаковых наименований товара, их количество должно суммироваться в одном ключе словаря.

Введенные данные нужно сохранить в текстовый файл. Название файла может быть любым.

Информация по каждому товару должна быть записана в отдельной строке, а наименование и количество разделены двоеточием. Например, если было введено три товара и два из них оказались одинаковыми, в словаре будет два ключа — "Товар 1" и "Товар 2". Их значениями будут целые числа, например, 2 и 8. Так вот, для такого словаря, содержимое файла, в которое сохранен каталог, должно выглядеть так

Товар 1:2
Товар 2:8
Содержимое файла в редакторе не обновляется автоматически. То есть, чтобы увидеть изменения содержимого файла, нужно закрыть и снова открыть файл из левой панели.


Наше решение
catalog = {}
for i in range(3):
    name = input('Введите наименование товара')
    count = int(input('Введите количество товара'))
    # если ключ (наименование товара) уже есть
    # в словаре
    if name in catalog:
        # увеличиваем его значение на count
        catalog[name] += count
    else:
        # иначе создаем ключ со значением count 
        catalog[name] = count

# Каждый ключ и значение в каталоге нужно 
# # записать в файл через двоеточие            
with open('catalog.txt', 'w') as f:
    for i in catalog:
        f.write(i + ':' + str(catalog[i]) + '\n')
