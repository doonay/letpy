Для закрепления #31
Для написания программы из этого урока вы можете создать новый файл, либо очистить файл с программой из предыдущего урока. Главное, чтобы в файле был только тот исходный код, которой относится к текущей задаче.

Присвойте переменной с именем escape значение, в котором есть один перенос строки, двойная и одинарная кавычка, а также символ обратной косой черты \

Значение переменной escape выведите на экран функцией print. Обязательно проверьте, чтобы все необходимые символы отобразились в поле «Результат» при выводе на экран.



Решение:
В строку с помощью экранирования записаны поочередно символы одинарной кавычки, переноса строки, двойной кавычки и обратной косой черты

escape = "\' \n \" \\ какой-то текст"
print(escape)
Одинарную кавычку можно записать и без экранирования, так как строка ограничена двойными кавычками

escape = "' \n \" \\ какой-то текст"
print(escape)
Если строку ограничить одинарными кавычками, то без экранирования можно записать двойную кавычку

escape = '\' \n " \\ какой-то текст'
print(escape)
