Алгоритм и обмен значений двух переменных #28
В этом уроке нужно написать программу, которая обменивает две переменные значениями.

Для этого вы можете создать новый файл, любо очистить файл с программой из предыдущего урока. Главное, чтобы в файле был только тот исходный код, которой относится к текущей задаче.

Прежде чем начинать писать код, необходимо разработать алгоритм. Запомните это слово, оно точно пригодится не только в программировании, но и в жизни. Алгоритм — это последовательность шагов или операция для решения какой либо задачи.

Так вот, алгоритм для решения этой задачи будет выглядеть так:

Шаг 1

Присвоить переменной с именем a строку, полученную с помощью функции input. Второй переменной b присвойте другую строку, также полученную от пользователя.

Шаг 2

Создать дополнительную переменную, назвать ее например tmp и присвоить ей значение переменной a

Шаг 3

Переменной a присвоить значение переменной b

Шаг 4

Переменной b присвоить значение дополнительной переменной tmp. Это значение равно старому значению переменной a

Шаг 5

Убедиться, правильно ли вы все сделали. Для этого можно вывести значения переменных на экран. Хотя это и необязательно для прохождения задачи.

В последующих задачах обязательно пробуйте разрабатывать алгоритм самостоятельно. Например, можете записывать шаги на листке бумаги. Это помогает раздробить крупную или сложную задачу на более мелкие и простые части.

И еще, в Питоне есть более красивый способ обменять переменные значениями, но о нем будет рассказано немного позже. В этом же уроке важно понять, что код «в лоб»

a = b
b = a
не сработает, потому что код выполняется последовательно, и при выполнении первой строки, значение a будет затерто значением b. То есть, после выполнения такого кода и в a и b будет храниться одно и тоже значение.




a = input("Введите a")
b = input("Введите b")
tmp = a
a = b
b = tmp
