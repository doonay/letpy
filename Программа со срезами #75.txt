Программа со срезами #75
Напишите программу, которая записывает строку, введённую пользователем в переменную string. Программа должна проверить длину строки. Если длина строки больше либо равна пяти — программа должна вывести первые три и последние три символа введенной строки. Иначе — программа должна вывести надпись «Ошибка»


Наше решение
string = input("Введите что-нибудь")
if len(string) >= 5:
    print(string[:3], string [-3:])
else:
    print("Ошибка")
