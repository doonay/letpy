Программа с else #48
Напишите программу для проверки пароля, введённого пользователем.

В программе должна быть объявлена переменная с именем password — в ней будет храниться правильный пароль. Эта переменная будет строкой.

Значение второй переменной с именем user_password, должно быть получено с помощью функции input. Переменная user_password тоже должна быть строкой.

Если пользователь ввел правильный пароль, программа должна выводить надпись «Входите». Иначе — «Доступ запрещён». Для сравнения введённого и правильного пароля используйте оператор сравнения ==




Наше решение
password = 'letpy'
user_password = input("Введите пароль")
if user_password == password:
    print("Входите")
else:
    print("Доступ запрещен") 
Если запустить такую программу и ввести пароль «letpy» — программы выведет текст «Входите». Если ввести неверный пароль, то будет выведен текст «Доступ запрещен».
