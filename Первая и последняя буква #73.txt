Первая и последняя буква #73
Напишите программу, которая записывает строку, введённую пользователем в переменную string. Программа должна проверить длину строки. Если длина строки больше либо равна двум — программа должна вывести первый и последний символ этой строки. Иначе — программа должна вывести надпись «Ошибка»


Наше решение
С переменной для хранения длины строки

string = input("Введите любой текст")
string_length = len(string)
if string_length >= 2:
    print(string[0], string[-1])
else:
    print("Ошибка")
