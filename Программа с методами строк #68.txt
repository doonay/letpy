Программа с методами строк #68
Напишите программу, в которой переменной string присвоено значение, введённое пользователем. Тип переменной string — строка.

Программа должна преобразовать строку, введённую пользователем в верхний регистр с помощью метода upper и вывести на экран преобразованную строку.

В этом уроке для решения можно объявлять дополнительные переменные и использовать их по своему усмотрению.



Наше решение
Эту задачу можно решить так

string = input("Введите что-нибудь")
print(string.upper())
либо так, сохранив строку, приведенную к верхнему регистру

string = input("Введите что-нибудь")
string = string.upper()
print(string)
