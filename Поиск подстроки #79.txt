Поиск подстроки #79
Для поиска подстроки в строке есть метод find. Подстрока для поиска — единственный аргумент для вызова этого метода. Результатом вызова find будет позиция, в которой Питон первый раз нашел подстроку. Номера позиций начинаются с 0. Если подстрока не найдена, вызов find вернет −1.

Программа, которую вы напишете в этом уроке, должна получить строку от пользователя с помощью input. Если в строке найден символ #, программа должна вывести часть строки до этого символа. Иначе — программа должна вывести на экран исходную строку.

Наше решение
В переменную sharp_index запишется индекс символа решетки, если он будет найден методом find. Если символа решетки в строке не будет, в переменную sharp_index будет записано значение -1

text = input("Введите любой текст")
sharp_index = text.find('#')

if sharp_index != -1:
    print(text[:sharp_index])
else:
    print(text)
То есть, если значение sharp_index не равно -1, решетка в строке есть и надо вывести символы от начала строки и до индекса найденной решетки. Иначе нужно вывести строку на экран целиком.
