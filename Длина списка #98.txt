Длина списка #98
Количество элементов списка, как и длину строки, возвращает функция len.

Напишите программу, в которой переменной string присвоено значение, введённое пользователем. Тип переменной string — строка. Программа должна вывести на экран количество слов во введенной пользователем строке. Для решения используйте метод строки split(' '), который разделит строку по символу пробела и вернет список слов.

В этом уроке для решения можно объявлять дополнительные переменные и использовать их по своему усмотрению.


Наше решение
string = input("Введите что-нибудь")
words = string.split(" ")
print(len(words))
