В нижний регистр! #69
Эта программа должна делать почти то же самое, что и предыдущая. В общем, напишите программу, в которой переменной string присвоено значение, введённое пользователем. Тип переменной string — строка.

Программа должна преобразовать строку, введённую пользователем в нижний регистр с помощью метода lower и вывести на экран преобразованную строку.

В этом уроке для решения можно объявлять дополнительные переменные и использовать их по своему усмотрению.


Наше решение
Эту задачу можно решить так

string = input("Введите что-нибудь")
print(string.lower())
либо так, сохранив строку, приведенную к нижнему регистру

string = input("Введите что-нибудь")
string = string.lower()
print(string)
