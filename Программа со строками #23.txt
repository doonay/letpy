Программа со строками #23
Для написания программы из этого урока вы можете создать новый файл, либо очистить файл с программой из предыдущего урока. Главное, чтобы в файле был только тот исходный код, которой относится к текущей задаче.

Присвойте переменной с именем string любой текст. То есть string должна быть объявлена с помощью кавычек.

Следующей строкой программы напечатайте значение переменной функцией print.



string = "Какой-то текст"
print(string)
