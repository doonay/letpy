Еще одна программа #38
Для написания программы из этого урока вы можете создать новый файл, либо очистить файл с программой из предыдущего урока. Главное, чтобы в файле был только тот исходный код, которой относится к текущей задаче.

Переменной a присвойте результат выражения 55 + 105. Переменной b присвойте результат конкатенации строки "Сумма=" и преобразованной в строку переменной a.

Переменная a должна остаться целым числом. Преобразование числа в строку можно сделать с помощью вспомогательной переменной и функции преобразования типа str. Вспомогательную переменную можете назвать как хотите.

Переменную b выведите на экран.

Кстати, если у вас слишком много открытых файлов, можно закрыть файлы с программами из предыдущих уроков. Они все равно останутся доступными для открытия из панели слева и никуда не пропадут.




Есть два пути решения этой задачи. С промежуточной переменной

a = 55 + 105
string_a = str(a)
b = "Сумма= " + string_a
print(b)
и без промежуточной переменной

a = 55 + 105
b = "Сумма= " + str(a)
print(b)
