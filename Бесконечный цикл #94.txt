Бесконечный цикл #94
В этом уроке вам надо будет усовершенствовать предыдущую программу. Квадрат должен «лететь» от левого края к правому краю холста. Как только квадрат коснется правого края, он должен начать двигаться в обратную сторону. Коснувшись левого края, квадрат снова должен «полететь» в правую сторону. В общем, квадрат должен летать туда-сюда до тех пор, пока кто‑нибудь не нажмет кнопку 

Вместо цикла, который увеличивает x от 0 до 350, должен быть «бесконечный цикл». У этого цикла есть своя страница в Википедии.

На языке Python бесконечный цикл выглядит так:

while True:
    pass  # Тут может быть какой-то полезный код
Кроме переменной x в программе должна быть переменная delta, которая в начале должна быть равна 2. Каждую итерацию переменная x должна изменятся на эту переменную delta. То есть

x = x + delta
Также в каждой итерации должна быть проверка: если x стал меньше 0 или больше 300 — знак delta должен меняться на противоположный

delta = -delta


Наше решение
В зависимости от знака переменной delta координата x то увеличиваться, то уменьшаться. Таким образом квадрат будет сначала лететь из левой стороны в правую (x увеличивается), а потом из правой стороны в левую (x уменьшается)

import canvas

x = 0
delta = 2
while x < 350:
    if x < 0 or x > 300:
        delta = -delta
    canvas.clear()
    canvas.fill_rect(x, 100, 50, 50)
    canvas.draw()
    x = x + delta
